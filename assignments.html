Midterm 506: Amazon Movie Review Prediction

This project involves building a predictive model to classify Amazon movie reviews by star ratings. Using a Linear Support Vector Classifier (LinearSVC) and feature engineering techniques like TF-IDF text vectorization and metadata analysis, the model achieved a Kaggle public score of 0.64830 and a private score of 0.64952. This project demonstrates the effectiveness of feature engineering in text classification without deep learning.
